# 🧮 Calculator Web App

This project is a **modern calculator web application** built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations and includes a clean UI with a **dark/light theme toggle**.

---

## 🚀 Features

* ➕ Basic arithmetic operations (+, −, ×, ÷)
* 🧠 Real-time calculation using JavaScript
* ⌫ Backspace functionality
* 🧹 Clear screen option
* 🌙 Dark / Light mode toggle
* 🎨 Smooth UI with animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
Calculator/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

* User clicks buttons to input numbers and operators
* Expression is displayed on screen
* Clicking **=** evaluates the expression
* Backspace removes the last character
* Clear button resets the display
* Theme toggle switches between dark and light mode

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start calculating 🧮
